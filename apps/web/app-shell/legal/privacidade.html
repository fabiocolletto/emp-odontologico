Política de Privacidade

O OdontoFlow trata dados necessários para autenticação, operação clínica e segurança da conta com base em princípios de minimização e proteção.

  • Dados de login são utilizados para autenticação e controle de sessão.
  • Dados operacionais são usados para execução das funcionalidades contratadas.
  • O usuário pode solicitar revisão e atualização de informações cadastradas.

Versão desta política: 1.0 — vigente em 28/04/2026.
